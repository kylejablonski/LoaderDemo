# LoaderDemo
Sample app which shows the use of a non-exported content provider backed by a SQLiteDatabase



**In this demo:**
- Content Provider 
- SQLiteDatabase
- RecyclerView.ItemDecoration 
- RecyclerView use
- Fab use
- Snackbar use
- Loader use
